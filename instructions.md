# MTD Champaign Urbana Live Dashboard

- I want to use the newly available developer api for MTD champaign urbana.
- The homepage should be a live map view of all the vehicles running in champaign urbana.
- The theme and animation should be exactly like Mini Metro, with the same color scheme and bus styles.
- It should be fully responsive and should work on both desktop and mobile devices, but for the first iteration we will focus on Desktop only.
- Dark mode and White mode should be supported and toggle should be present in the top right corner.
- Extremely minimal themes. No fluff text, no borders, no padding, no nothing. Just the minimal map and the buses.
- Color of routes is the color of the buses. It should be exactly like Mini Metro.
- All the routes running should be visible on the map, in their specific color.
